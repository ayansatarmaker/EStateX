# 🏡 EstateX – Property Listing Website

## 📖 Description
EstateX is a responsive **Property Listing Website** built using **HTML, CSS, and JavaScript**. The website provides a clean and intuitive interface for users to browse property listings with essential details such as property images, location, price, and description. It is designed with a mobile-friendly layout to ensure a seamless experience across different screen sizes.

The project demonstrates core front-end development concepts, including responsive web design, structured layouts, and interactive user interfaces using vanilla JavaScript.

## ✨ Features
- 📱 Fully responsive design for desktop, tablet, and mobile devices.
- 🏠 Browse property listings with images and detailed information.
- 🎨 Clean and modern user interface.
- ⚡ Interactive elements implemented using JavaScript.
- 🧭 Easy-to-use navigation and user-friendly experience.
- 📂 Well-organized project structure for easy maintenance.

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Project Objective
The goal of EstateX is to develop a responsive and visually appealing property listing platform that showcases front-end web development skills while providing an engaging user experience.
